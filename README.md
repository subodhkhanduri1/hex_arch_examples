# hex_arch_examples
Hexagonal Architecture examples in Ruby

## Setup

- Clone the repository
```
git clone https://github.com/subodhkhanduri1/hex_arch_examples.git
```
- Install Bundler
```
gem install bundler
```
- Install Dependencies
```
bundle
```

## Running Rspec unit tests

```
rake
```

## Running IRB console

rake console
