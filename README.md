# hex_arch_examples
Hexagonal Architecture examples in Ruby

## Setup

- Clone the repository
```
git clone https://github.com/subodhkhanduri1/hex_arch_examples.git
```
- Install Bundler
```
gem install bundler
```
- Install Dependencies
```
bundle
```

## Running Rspec unit tests

Run once:
```
rake
```

Run on update using guard
```
guard
```

## Running IRB console

rake console
